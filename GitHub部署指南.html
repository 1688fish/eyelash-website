2创建新仓库

登录GitHub后，创建一个新的仓库：

➕ 创建新仓库

填写仓库信息：

  1. Repository name: eyelash-website
  2. Description: False Eyelash World Factory Website
  3. Public/Private: 选择 Public（公开）
  4. 不要勾选 "Add a README file"
  5. 点击 Create repository
↓

3上传网站文件

创建仓库后，GitHub会显示一个页面。请按照以下步骤操作：

方法A：使用命令行（推荐）

在您的电脑上打开命令提示符（CMD），然后依次运行以下命令：

cd C:\Users\Administrator\lobsterai\project\eyelash-website\deploy-v2
git remote add origin https://github.com/您的用户名/eyelash-website.git
git branch -M main
git push -u origin main

注意：将 您的用户名 替换为您的GitHub用户名

方法B：使用网页上传

  1. 在GitHub仓库页面，点击 "uploading an existing file"
  2. 将 deploy-v2 文件夹中的所有文件拖到页面中
  3. 点击 "Commit changes"
↓

4启用GitHub Pages

文件上传完成后，启用GitHub Pages：

  1. 在仓库页面，点击 Settings（设置）
  2. 在左侧菜单找到 Pages
  3. 在 "Source" 下拉菜单中选择 main 分支
  4. 文件夹选择 / (root)
  5. 点击 Save

等待1-2分钟，页面会显示您的网站地址：

https://您的用户名.github.io/eyelash-website/

🎉 部署完成！

您的网站现在已经上线了！
网址格式：https://您的用户名.github.io/eyelash-website/

这个网站将永久免费托管在GitHub上。
